One of the most common hieroglyphs, snake, was used in Egyptian writing to stand for a sound like the English ‚ü®J‚ü©, because the Egyptian word for "snake" was djet. It is speculated by many that Semitic people working in Egypt adapted hieroglyphics to create the first alphabet, and that they used the same snake symbol to represent N, because their word for "snake" may have begun with that sound. However, the name for the letter in the Phoenician, Hebrew, Aramaic and Arabic alphabets is nun, which means "fish" in some of these languages. The sound value of the letter was /n/‚Äîas in Greek, Etruscan, Latin and modern languages.

Use in writing systems[edit]

‚ü®n‚ü© represents a dental or alveolar nasal in virtually all languages that use the Latin alphabet, and in the International Phonetic Alphabet. A common digraph with ‚ü®n‚ü© is ‚ü®ng‚ü©, which represents a velar nasal in a variety of languages, usually positioned word-finally in English. Often, before a velar plosive (as in ink or jungle), ‚ü®n‚ü© alone represents a velar nasal. In Italian and French, ‚ü®gn‚ü© represents a palatal nasal /…≤/. The Portuguese and Vietnamese spelling for this sound is ‚ü®nh‚ü©, while Spanish and a few other languages use the letter ‚ü®√±‚ü©. In English, ‚ü®n‚ü© is generally silent when it is preceded by an ‚ü®m‚ü© at the end of words, as in hymn; however, it is pronounced in this combination when occurring word medially, as in hymnal.

‚ü®n‚ü© is the sixth most common letter and the second-most commonly used consonant in the English language (after ‚ü®t‚ü©).[2]

Other uses[edit]

In mathematics, the italic form n is the common symbol for a variable quantity, especially one which represents an integer. is often used to refer to the set of natural numbers.

In statistics, the italic form n is used to denote the number of observations or replicates included in a statistical sample.

In chemistry, N represents the element nitrogen. In biochemistry, N is the one-letter code symbol for the amino acid asparagine. It is also the symbol for the SI unit of force, the newton.
